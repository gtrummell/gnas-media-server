# GNAS 415 Linuxserver.io Portainer Stack

This directory contains the docker-compose.yml and supporting files needed to construct
a Portainer Stack running the following applications:
  - bazarr
  - hydra2
  - jackett
  - jellyfin
  - kodi_headless
  - lidarr
  - mysql
  - openvpn_as
  - plex
  - radarr
  - sabnzbd
  - sonarr
  - transmission
  - sabnzbd_exporter
  - transmission_exporter
